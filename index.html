SMART STREET LIGHTING SYSTEM


An IoT-based street lightning system which uses nodemcu and blynk notification


Introduction

Our mini-project addresses the challenges like unnecessary power consumption and human presence through the implementation of a Smart Street Lighting System using the ESP8266 microcontroller, infrared (IR) sensors, and light-dependent resistor (LDR) sensors. The integration of these technologies aims to create an intelligent, adaptive, and energy-efficient lighting infrastructure for urban environments.

Components Used

NodeMCU ESP8266
IR-Sensor
LDR-sensor
LED
Resistor


Implementation




STEP 1 : To implement the project the following connections should be made

  • OUT from IR sensor 1 to D0 of ESP8266
  • OUT from IR sensor 2 to D1 of ESP8266
  • OUT from IR sensor 3 to D2 of ESP8266
  • OUT from LDR sensor to A0 of ESP8266
  • GND from IR sensor 1,2,3 and LDR sensor to GND of ESP8266
  • VCC from IR sensor 1,2,3 and LDR sensor to VIN of ESP8266
  • ANODE of LED1 to D5 of ESP8266
  • ANODE of LED2 to D6 of ESP8266
  • ANODE of LED3 to D7 of ESP8266
  • CATHODE of LED 1,2,3 to GND of ESP8266
  • One end of resistor to A0 and the other end to the GND of ESP8266

STEP 2:

  • Install Arduino IDE from https://www.arduino.cc/en/software
  • Open the IDE and copy the code given in the code section
  • To install ESP8266 board
    • Go to File -> Preferences and paste http://arduino.esp8266.com/stable/package_esp8266com_index.json in Additional Boards Managers URLs
    • Once the installation is complete go to Tools -> Board -> ESP8266 Boards -> NodeMCU 1.0 (ESP-12E Module)

STEP 3:

  • Connect the ESP8266 to a pc using a USB cable
  • Open the IDE and select the port in Tools
  • Upload the program

Set up Blynk App for the project


Blynk is a popular Internet of Things (IoT) platform that enables users to build and control IoT projects easily. It simplifies the process of creating connected devices by providing a user-friendly interface and a powerful cloud infrastructure. Blynk supports a variety of hardware platforms and communication protocols, making it versatile and widely used in the maker and IoT community


Steps to follow in blynk:

  • Install the blynk app from https://blynk.io/
  • Create a new project
  • Enter the authentication code sent in the code in IDE
  • Add terminal and select PIN as V10
  • Run the project

Working


The system is designed to dynamically adjust the intensity of streetlights based on real-time environmental conditions, human presence, and ambient light levels. The ESP8266 controls the intensity of the street lights according to the readings of the LDR sensor and IR sensors. LDR sensor recognises whether its day or night and according to it esp8266 sets the street lights in OFF or LOW INTENSITY states. When a vehicle approaches a specific street light the IR sensor senses the movement and the ESP8266 sets that particular street light in HIGH state.
